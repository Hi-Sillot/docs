```sh
npm install -g yo generator-code
```


* 使用 `yo code`命令生成一个新的扩展。

pnpm i -D @vscode/vsce 安装到插件项目

在 package.json 添加字段

```sh
"vsce": {
    "baseImagesUrl": "https://my.custom/base/images/url",
    "dependencies": false,
    "yarn": false
  }
```

编辑插件项目的 Readme.md 内容

运行 pnpm vsce package 打包
